# MODULE 16 - Web Sockets 2.0  
Startbestanden van de server voor Boter-Kaas-Eieren  
  
# Inleiding
Tijdens de instructiemomenten gaan we samen een Multiplayer Boter-Kaas-Eieren spel maken. Daarvoor gaan we tijdens de instructiemomenten de server bouwen in ***PHP***. Daarbij maken we gebruik van de command-line tool **Composer**. Composer is de dependency manager voor de programmeertaal PHP. Deze tool gaan we o.a. gebruiken om onze code te structureren en extra packages in ons project te installeren voor gebruik in de server.  

# Deze map
Deze map bevat alle bronbestanden om te starten met het bouwen van de server.
